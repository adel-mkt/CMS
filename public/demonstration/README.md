## Fonctionnalités principales

### Pages du projet

1. **Page d'accueil (`index`)**
   - Permet de **saisir les notes** des étudiants.
   - On peut **ajouter autant de notes que nécessaire**.
   - Les notes sont classées selon :
     - **Contrôle continu**
     - **Partiel**
     - **Examen**
   - Une fois les notes envoyées, le serveur calcule :
     - La **note de chaque matière**, en utilisant les coefficients enregistrés côté serveur pour chaque type de note.
     - La **moyenne générale** de l'étudiant pour le bloc (1 bloc deja pré entré pour la demo).

2. **Page des résultats**
   - Affiche la **moyenne générale** et les résultats détaillés.
   - Indique si l’étudiant est **admis** ou **non admis** selon la moyenne.
